# Small annotation tools for text image alignment


## Preparation

Install PyQt5
```bash
pip install PyQt5
```

Install jsonlines
```bash
pip install jsonlines
```

Install nltk and corpus
```python
import nltk
nltk.download('punkt')
```

## Run annt script

1. Please change the annt path @line 163 of ```pyqt_annt.py```

2. run the following
```bash
python pyqt_annt.py 
```
